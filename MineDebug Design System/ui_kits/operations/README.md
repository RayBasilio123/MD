# MineDebug · Operations UI Kit

> ⚠️ **Disclaimer.** A MineDebug não forneceu codebase, Figma ou screenshots de produto. Este UI kit é uma **proposta de aplicação da identidade visual** (cores, tipografia, motion, iconografia do design system) em uma interface plausível para o domínio — não uma recriação de produto existente. Substituir por componentes reais quando disponíveis.

## O que está aqui

- `index.html` — dashboard de operações em modo escuro (sala de controle)
- `Topbar.jsx`, `Sidebar.jsx` — chrome de navegação
- `MetricCard.jsx`, `FleetTable.jsx`, `AlertList.jsx`, `Sparkline.jsx` — componentes de telemetria
- `EquipmentRow.jsx` — linha individual de caminhão / equipamento

## Tela representada

**Painel Operações · Frota.** Lista caminhões fora-de-estrada com consumo de diesel em tempo real, status de sensor, dosagem de aditivo, e CO₂ evitado vs. baseline. É a tela que um técnico de monitoramento veria às 3h da manhã num galpão climatizado.

## Densidade

Linhas de 40px, tipografia mono em todos os números, hairlines em vez de cards aninhados. Modo escuro primário.
