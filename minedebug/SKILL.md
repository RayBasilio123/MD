If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.
If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick reference

- **Tokens:** `colors_and_type.css` — import this in any new HTML file you build.
- **Logo:** `assets/logo.svg` (vector) and `assets/logo.jpeg` (raster fallback).
- **Field photography:** `assets/equipamento.jpeg`, `assets/instalacao-simples.jpeg`, `assets/vista-equipamento.jpeg`, `assets/robson-csn.jpeg`.
- **UI kit:** `ui_kits/operations/` — reference dashboard with React components.
- **Icons:** Lucide via CDN (`https://unpkg.com/lucide@latest/dist/umd/lucide.js`), 2px stroke.

## Brand essence

Industrial Brazilian mining-tech. Black + neon green (#19E27A). Mono numerals. Small radii. No emoji in product. Portuguese (pt-BR) primary. Tone: technical, measurable, direct — never marketing-ês.
